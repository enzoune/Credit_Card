# 💳 Credit Card UI (Interface de Cartão de Crédito)

> Um componente de interface elegante e estruturado que recria a experiência visual de um cartão de crédito, focando em organização de dados e semântica HTML.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma interface de **Cartão de Crédito** realista, demonstrando competência na disposição de elementos complexos em containers fixos e na hierarquia de informações.

O foco do desenvolvimento foi a clareza visual, garantindo que elementos como o chip, o logotipo da bandeira e os dados do portador estivessem posicionados de forma profissional.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura Semântica:** Uso correto da tag `<header>` para isolar os elementos de identificação do cartão (Logo e Chip EMV).
* **Hierarquia de Informação:** Utilização de diferentes níveis de títulos (`h5` e `h6`) para diferenciar rótulos de dados (ex: "Card Number") dos valores reais, melhorando a escaneabilidade.
* **Organização Modular:** Estruturação através de classes como `.card-details` e `.name-number`, facilitando a manutenção do layout e futura aplicação de estilos responsivos.
* **Fidelidade Visual:** Inclusão de recursos gráficos como ícones de atalho e logos, simulando uma interface de pagamento real.
* **Boas Práticas de Meta-Dados:** Configuração de charset, viewport e compatibilidade de navegador, demonstrando atenção aos padrões web modernos.
* **Separação de Responsabilidades:** O código mantém uma separação estrita entre a marcação (`index.html`) e a estilização externa (`css/style.css`).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Marcação estrutural e semântica.
* **CSS3:** Gerenciamento de layout, tipografia e posicionamento de elementos.
